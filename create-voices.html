Tạo File Giọng Nói Cho VTuber

Hướng dẫn:

  1. Nhấn vào nút "Tạo Giọng Nói" để tạo âm thanh mẫu
  2. Nghe thử âm thanh bằng cách nhấn nút play
  3. Nếu bạn thích âm thanh đó, nhấn "Tải Xuống" để lưu file
  4. Đặt file âm thanh đã tải xuống vào thư mục "voices" trong dự án của bạn
